一个：表示的是伪类，两个：：表示的是为伪元素

div::first-letter表示的是对div的第一个单词的第一个字进行css

div::first-line表示对div第一行进行css

div::selection表示对div中鼠标选中的内容进行css

input::placeholder为对placeholder进行css
span::after对span内容末尾进行css，span::before对span内容开头进行css，content表示想添加的内容

在元素上，content 的初始值为 ‘normal’。在:before和:after上，如果指定了content 的初始值为 ‘normal’，则计算为 ‘none’ 。content 的值设置为 ‘none’ 不会生成伪元素。所以:before和:after才需要指定一个看似无意义的 content: ""; 来初始化content的值。

添加伪类元素是块级元素。不能设置宽高。要转换为行级元素 display:block
