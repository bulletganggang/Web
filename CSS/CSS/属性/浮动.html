float: left 开启浮动

消除浮动的影响

在最外面容器设置一个， .outer::after { content: ''; display: block; clear: both; (看情况加)height: 100px; }

请注意，使用了float之后，如果原先没有设置高度，则父级盒子的高度变为0了

可以理解为，原先的父级由内容撑开，但子元素浮动之后，脱离文档流，父级没有内容撑开，高度自动变为0
