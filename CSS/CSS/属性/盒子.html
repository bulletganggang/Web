盒子由四部分组成，content、padding、border、margin

content可以选择min\max width\height

padding可以选择left right top bottom，也可以选择复合属性

margin和padding:一个值表示四个方向，两个值表示上下、左右，三个值表示上、左右、下，四个值表示上右下左

行内元素的padding左右没问题，上下会出界

border也同理，有left right top bottom的border-X-width,border-X-color,border-X-style

当然也可以使用复合属性,border-X:

margin的值可以设置为auto，margin:0 auto可以水平居中，需要是块级元素

overflow可以处理文本内容溢出，一般设置为hidden和auto
